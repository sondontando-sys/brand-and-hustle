AI-powered branding + hustle tools for youth and small businesses.
# BRAND&HUSTLE

**Mission:** Empower African youth and small businesses with AI-powered branding, skills, and income tools.

**Features:**
- AI Branding Studio: Logo, story, social posts
- Skill Builder: Micro-courses on AI, IT, and sales
- Marketplace: Freelance gigs and client connections
- Radio Hub: Publish audio segments and build community
- Monetization Engine: Subscriptions, payouts, affiliate rewards

**Tech Stack:**
- Frontend: React Native
- Backend: Node.js (NestJS), PostgreSQL
- AI: Text + image generation, proposal assistant
- Hosting: AWS or GCP
